Performance visualizer - unstable development version

Back to portal   Development Version of experiment for Google Chrome. Not intended for other browsers. Audio plays only in Google Chrome. May run with some versions of Firefox, albeit very slowly.
Executes a long running process, which periodically yields the processor to allow asynchronous handlers to run. The long running process submits sounds to be played. The audio machinery does not posts sounds to the audio machinery at every submitted sound, but only when the audio machinery is about to run out of sounds to play.
